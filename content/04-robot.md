Both Simple Simon and Standard League robots must be built from LEGO parts only. The key difference is which sensors and wheels are allowed. Every robot must pass scrutineering before competing — referees check size, weight, ball capture zone, and construction.

| Specification | Simple Simon | Standard |
|---|---|---|
| Construction | LEGO only | LEGO only |
| Max diameter | 220 mm | 220 mm |
| Max height | 220 mm | 220 mm |
| Max weight | 1.0 kg | 1.0 kg |
| Ball capture zone | 30 mm max | 30 mm max |
| IR sensor | Approved list only | Approved list only |
| Gyro / compass | **Not allowed** | Allowed |
| Omni wheels | **Not allowed** | Allowed (80 mm max, LEGO axle) |
| Dribbler | Not allowed | Not allowed |
| Power source | LEGO rechargeable battery | LEGO rechargeable battery |
| Participation limit | **2 years** | No limit |

> **Handle requirement:** Every robot must have a stable, easily accessible handle at least 50 mm above the highest structure. Cable ties make a strong, lightweight handle. The handle height does not count toward the 220 mm robot height limit.

> **Approved sensors:** Only sensors on the official list are allowed. Check the current list at [robocupjunior.org.au/soccer/sensors](https://www.robocupjunior.org.au/soccer/sensors/).

> **The 220 mm cylinder test:** Your robot must rotate freely inside a 220 mm diameter cylinder with no resistance. Build to 210 mm to allow for tolerance. Any moving parts must be fully extended during inspection.
